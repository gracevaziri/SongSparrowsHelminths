This file describes the variables in 
Temp_data_for_figure2.csv
Which was used in 
Anthelmintic drugs modulate the acute phase immune response but not the microbiota in wild Song Sparrows
"by Grace J. Vaziri, Michelle A. Jusino, Jon A. Palmer, Matthew T. Brewer and James S. Adelman"


hours_since_inj = the number of hours that have passed since a bird was injected with LPS
tch_avs = the bird's average body temperature during a particular half hour interval
tch_ses = the standard error for mean temperature
"trt = the anthelminthic and lps treatment status, separated by a ""_"""
fen = the anthelmintic treatment status (Y or N)
lps = the LPS treatment status (Y or N)
meanANDse = the mean temperature for each half hour plus the standard error
meanMINUSse = the mean temperature for each half hour minus the standard error
